# 冯健彤的个人网站

> 华中科技大学计算机科学与技术学院学生个人主页。

使用纯 HTML/CSS/JavaScript 构建的个人网站，通过 GitHub Pages 自动部署。

## 技术栈

- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **部署**: GitHub Pages (GitHub Actions 自动部署)
- **CI/CD**: `static.yml` workflow — push 到 main 分支即自动发布
